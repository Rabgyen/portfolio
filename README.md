# Portfolio Website

Personal portfolio website for **Rabgyen Moktan**, built with HTML, CSS, and JavaScript.
It highlights profile details, skills, projects, current work-in-progress, and a contact form.

## Live Demo

Visit the live site here:

https://rabgyen.github.io/portfolio/

## About This Portfolio

This portfolio is designed as a single-page personal website to showcase:

- Intro/profile section with social links and CV download
- About section with background and hobbies
- Skills section with technology proficiency
- Project showcase with GitHub and live links
- "Working On" section for ongoing projects
- Contact form powered by Web3Forms

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Boxicons (icon library)
- Google Fonts

## Running Locally

1. Clone the repository:

	```bash
	git clone https://github.com/Rabgyen/portfolio.git
	```

2. Open the project folder:

	```bash
	cd portfolio
	```

3. Open `index.html` in your browser.
